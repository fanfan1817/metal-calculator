# 金属建材计算器 - 规格文档

## 1. 项目概述

- **项目名称**：金属建材计算器
- **类型**：Windows桌面应用程序
- **核心功能**：提供四个常用金属建材重量计算公式，支持单位标注
- **技术栈**：Python 3 + Tkinter（内置库，无需额外安装）

## 2. 公式定义

### 2.1 圆管公式
- **标题**：圆管重量计算
- **描述**：直径（毫米）-厚度（毫米）×厚度（毫米）×厚度（毫米）×0.02466=1米的重量
- **按顺序计算**（不使用数学优先级）
- **参数**：
  - 直径（毫米）
  - 厚度（毫米）
- **输出**：1米重量，单位：公斤（kg）
- **计算过程**：(直径 - 厚度) × 厚度 × 厚度 × 0.02466

### 2.2 方管公式
- **标题**：方管重量计算
- **描述**：周长（毫米）÷3.14-厚度（毫米）×厚度（毫米）×0.02466=1米的重量
- **按顺序计算**
- **参数**：
  - 周长（毫米）
  - 厚度（毫米）
- **输出**：1米重量，单位：公斤（kg）
- **计算过程**：（（周长 ÷ 3.14）- 厚度）× 厚度 × 0.02466

### 2.3 丝径公式
- **标题**：丝径重量计算
- **描述**：长度（米数）×丝径（毫米）×丝径（毫米）×0.00617=整个重量
- **按顺序计算**
- **参数**：
  - 长度（米）
  - 丝径（毫米）
- **输出**：总重量，单位：公斤（kg）
- **计算过程**：长度 × 丝径 × 丝径 × 0.00617

### 2.4 法兰盘公式
- **标题**：法兰盘重量计算
- **描述**：7.85×边长（米）×另一边边长（米）×厚度（毫米）=整个重量
- **按顺序计算**
- **参数**：
  - 边长1（米）
  - 边长2（米）
  - 厚度（毫米）
- **输出**：总重量，单位：公斤（kg）
- **计算过程**：7.85 × 边长1 × 边长2 × 厚度

## 3. UI设计

### 3.1 窗口布局
- 窗口标题：金属建材计算器
- 窗口尺寸：480 × 520 像素
- 使用Notebook组件（标签页）分为4页

### 3.2 每个标签页布局
```
┌─────────────────────────────────┐
│         [公式标题]              │
│         [简要描述]              │
├─────────────────────────────────┤
│  参数1名称: [输入框]  单位      │
│  参数2名称: [输入框]  单位      │
│                                 │
│         [计算按钮]              │
│                                 │
│  ┌─────────────────────────┐   │
│  │      计算结果: XX kg     │   │
│  └─────────────────────────┘   │
└─────────────────────────────────┘
```

### 3.3 样式
- 字体：微软雅黑，标题14pt，标签10pt
- 按钮：蓝色背景，白色文字
- 结果区域：绿色边框，居中显示
- 输入框：白色背景，带边框

## 4. 功能需求

### 4.1 输入验证
- 只接受数字输入（整数或浮点数）
- 非法输入时显示错误提示
- 空输入时禁用计算按钮

### 4.2 计算
- 点击"计算"按钮执行计算
- 按公式顺序依次计算（不使用数学运算优先级）
- 结果保留4位小数

### 4.3 结果显示
- 大字显示计算结果
- 清晰标注单位（kg）
